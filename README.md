
# Educational Organization Service Management System

This project implements a centralized Service Management System for an Educational Organization using the **ServiceNow** platform. It streamlines student, faculty, and staff requests while improving service efficiency and transparency.

---

## 🎯 Objectives

* Centralize service requests
* Automate approval workflows
* Track incidents and SLAs
* Manage institutional assets
* Provide reports and dashboards

---

## 🛠️ Modules

### 1. Incident Management

* Report and track IT issues
* Auto-assignment to support teams
* SLA monitoring

### 2. Service Request Management

* Certificate requests
* ID card reissue
* Software installation
* Leave applications

### 3. Asset Management

* Track IT assets
* Allocate equipment
* Maintain asset records

### 4. Knowledge Base

* FAQs and troubleshooting guides
* Academic and IT policies

---

## 👥 User Roles

* **Student** – Raise and track requests
* **Faculty/Staff** – Submit service requests
* **IT Support** – Resolve incidents
* **Admin** – Configure and manage system

---

## 📊 Features

* Role-based access control
* Automated workflows
* Real-time dashboards
* Email notifications
* Audit tracking

---

## 🚀 Benefits

* Faster issue resolution
* Reduced paperwork
* Improved service visibility
* Better resource management
